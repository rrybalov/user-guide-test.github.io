Find & register a client

Two guides that work as a pair. You search first; you only register if the search finds no one. Done in order, they keep one person from ending up in the system twice.

Why the order matters. Every client lives in the county’s Master Client Index (MCI). If you register without searching, you can create a second record for someone who already exists — splitting their history. Search first, every time.
1

Search for a client

Always do this first. Searching tells you whether the person is already in the system so you don’t create a duplicate.
