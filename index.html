Search for and create a 302 petition

How to find an existing 302 petition and, only when needed, create a new one — written so you never accidentally create a duplicate for someone already in the system.

Why this matters

A 302 begins an involuntary emergency examination for someone in crisis. A person should have only one active petition — and theirs may already be in progress in another provider’s or a hospital’s bucket. Searching first, and calling the County Delegate whenever IRES prompts you, is how you keep one person’s case from being split across systems. You must always search before you can create a new petition.

How to search

Searching never changes anything — it’s always safe to run again.

Run a 302 search

Three steps, then read your result below.
  1. Open 302 Petitions and choose how to search.Check the box next to a criterion: Characteristics (name, date of birth, gender), Petition ID, Examining Facility, or Petition Status.
  2. Enter the required information, then select Search.You can combine criteria. Fields marked with an asterisk are required.
  3. Read your result and act on it.You’ll get one of three outcomes — see below.

The 302 Petitions search screen. Check a criterion, fill in its fields, and select Search.

What your search result means

Your search returns one of three outcomes. Each tells you whether to continue an existing petition or create a new one.

A · A matching petition is found

The 302 is already in your bucket.

Continue it

Select Show to open and continue that petition. Only select Create New Petition if you’ve confirmed this is not the right person or petition.

A result row appears. Use Show to continue it.

B · “Petition may exist — please call the County Delegate”

A petition might already be in progress elsewhere.

Call first

Don’t create a petition yet. Call the County Delegate at 412-350-4457 to check whether one is already in progress in another provider’s or a hospital’s bucket. If one exists, you likely don’t need a new one. Create a new petition only after the delegate confirms there isn’t.

This pop-up means: pause and call the delegate before creating.

C · “No Matches Found”

No existing petition for this person.

Create new

Select Create New Petition to start a new 302.

No match — safe to create a new petition.

If something goes wrong

The main risk is a duplicate petition. Here’s how to handle the common situations.

Fixing a mistake

Most situations are recoverable through the County Delegate.
Good to know. Running a search never creates or changes a record — only Create New Petition does. When in doubt, search again and call the delegate before you click it.
What to do
Created a new petition when one already existedRecoverable — call the County Delegate at 412-350-4457 right away. They coordinate across providers and hospitals and can resolve the duplicate.
Not sure whether the person already has a petitionCheck — search by Characteristics first. If you get “Petition may exist,” call the delegate before creating.
Opened the wrong petition with ShowRecoverable — opening it changes nothing. Go back to Search and run it again with the correct details.
Got “Petition may exist” and aren’t sure whether to proceedCheck — default to calling the delegate. Creating without checking is what produces a duplicate.
