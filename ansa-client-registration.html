Register a new client

How to add a client who isn’t in the system yet — and verify them against the county’s master index so you don’t create a duplicate.

Tool
ANSA
Last reviewed

Why this matters. Registration writes a new person into the Master Client Index (MCI). The Verify Individual step is your safety check: it compares what you typed against everyone already in MCI, so a person who exists under a slightly different spelling still gets caught before a duplicate is created.

Before you start

You’ve already run a client search and it came back “No Match Found.” The New button only becomes available after a search — that’s by design.

Steps

  1. Select New.From the search screen, after “No Match Found,” choose New to open a blank client record.
  2. Enter the core identifiers.First Name, Last Name, SSN, Date of Birth, and Gender.
  3. Select Verify Individual.This checks your entry against the Master Client Index.
  4. Handle the match result.If MCI returns a possible match of 95% or higher, you must select the existing person rather than create a new record. If you’re certain it’s a different person, the Service Desk can override. If there’s no match, continue.
  5. Complete the demographics.Add Race and Ethnicity to finish the person’s profile.
  6. Add an address.Choose New (or Edit), enter the address, and validate against Postal. If it won’t validate but you know it’s correct, use Save Without Verification.
  7. Select Save.The client is registered and ready to assess.

If something goes wrong

What to do
Verify Individual returns a 95%+ match but you believe it’s a different personCheck first — open the match and compare DOB and SSN. If it truly is someone else, contact the Service Desk for an override; don’t force a duplicate.
You registered someone who turned out to already existRecoverable — call the Service Desk (412-350-4357) to merge or correct the records.
The address won’t validate against PostalOption — if you’ve confirmed it’s right, use Save Without Verification rather than altering a correct address.
The New button is greyed outExpected — run a search first. New only enables after a search has run.
